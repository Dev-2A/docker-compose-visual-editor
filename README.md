# 🐳 Docker Compose Visual Editor

Docker Compose의 서비스, 네트워크, 볼륨을 노드 그래프로 시각화하고,
드래그앤드롭으로 편집하여 `docker-compose.yml`을 자동 생성하는 브라우저 기반 비주얼 에디터입니다.

> 🚧 개발 중입니다.

## 기술 스택

- React + Vite
- ReactFlow (노드 그래프)
- js-yaml (YAML 파싱/생성)
- Tailwind CSS (스타일링)
